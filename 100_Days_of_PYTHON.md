## Day 1
The `print()` statement:
```python
print('Hello World!')
```
The `print()` statement is used to print messages. You will use this a *lot*.

To `print()` multiple lines in one statement triple quotes are used.

```python
print('''
This will print as many lines of text as you need.

Seriously.
''')
```
